1) Hyper Text Transfer Protocol

2) Uniform Resource Locator

3) Domain Name System. takes human-readable URLs and converts them into IP addresses.

4) The query string allows you to pass key-value pairs into the URL, in the format

5) GET - get some data from the server
POST - send some data to the server

6) An HTTP request is a request from a client to a server which follows the HTTP protocol and requests some info like html, js, css etc.

7) An HTTP response is a response from a server to a client which follows the HTTP protocol and sends some info like html, js, css, etc.

8) Headers provide additional information about the request or the response.
Request headers: Host, User-Agent, Accept, Cookie, Cache-Control

9) 
-Your browser “resolves” the name into an IP address using DNS
-Your browser makes a request to that IP address, including headers
-The server sends a response
-The browser makes a DOM from that HTML, and finds any other resources needed
-The browser makes separate HTTP requests for those resources and receives response from the server for each
